# EMRI: Tests of General Relativity

## No-Hair Theorem Test

Kerr multipole moments: M_l + iS_l = M(ia)^l

Parametrized deviation: introduce quadrupole deviation delta_Q.
Three-frequency structure sensitive to delta_Q.

Typical LISA constraint: delta_Q < 10^{-3}-10^{-4} (SNR=30, 1yr)

## Kerr Symmetry Breaking (Muguruza & Sopuerta 2026, 2604.06053)

- Equatorial symmetry breaking: odd multipoles (M1, M3, ...)
- Axial symmetry breaking: non-axisymmetric components
- Constraints (SNR=30, 1yr): equatorial ~10^{-2}, axial ~10^{-3}

## ECO Tests

- No event horizon -> GW echoes
- Reflective boundary -> oscillating flux (Sago & Tanaka 2022)
- Different ISCO -> orbital frequency deviation
