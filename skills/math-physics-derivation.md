# 数学物理推导标准协议

面对公式推导或方程求解任务，**严格按以下顺序**工作，不得跳步。

## 1. 问题判型

先回答以下问题，再动手：

- 未知量是什么？自变量与参数是什么？
- 定义域是什么？边界条件/初值条件是什么？
- 方程类型：代数方程 / ODE / PDE / 特征值问题 / 积分方程 / 变分问题？
- 线性还是非线性？
- 是否存在：对称性、守恒量、规范自由度、可分离结构？

## 2. 结构分析

- 奇点在哪里？正则奇点还是不正则奇点？
- 哪些极限重要（$r\to\infty$，$r\to r_+$，$\omega\to 0$，等）？
- 是否适合无量纲化？是否适合提取 prefactor？
- 是否有小参数、大参数、渐近区、边界层？

## 3. 方法选择

**在推导前**，明确说明"为什么选这个方法"：

| 方法 | 适用条件 |
|------|----------|
| 分离变量 | 线性PDE，坐标系与边界匹配 |
| Frobenius 展开 | 正则奇点附近的级数解 |
| Fourier/Laplace 变换 | 无界域，平移不变性 |
| Green 函数 | 非齐次方程，已知边界条件 |
| Sturm-Liouville | 自伴算子，本征函数展开 |
| WKB | 短波极限，缓变势 |
| matched asymptotics | 边界层，多尺度 |
| null foliation + log-derivative | 黑洞扰动，谱方法正则化 |
| 扰动展开 | 小参数，近似解 |

## 4. 推导过程

- **分步写**，每步说明依据（哪条方程、哪个恒等式、哪个近似）
- 明确每个符号定义、每个替换的适用条件
- 避免大步跳跃；如有多个分支，说明为何选当前分支
- 每个近似写出适用范围（如"$r\gg M$ 时"）

## 5. 校验过程（必做）

推导后**主动**检查：

- [ ] 代回原方程是否成立？
- [ ] 维度/量纲是否一致？
- [ ] 极限是否合理（已知极限情形）？
- [ ] 特例是否退化到已知结果（如 $a\to 0$，$s\to 0$）？
- [ ] 边界条件是否满足？
- [ ] 如可行，做符号或数值残差检查

## 6. 物理解读

- 这个结果对应什么物理意义？
- 哪些项主导？哪些近似决定了解的结构？
- 结果在哪些参数区间可信？
- 哪些地方仍需数值验证？

## 审稿式复核清单

用于检查他人或自己的推导：

- [ ] 是否偷换符号（同一字母不同含义）？
- [ ] 是否漏条件（边界条件、收敛条件、适用范围）？
- [ ] 是否跳步（从A直接到C，跳过B）？
- [ ] 是否误用边界条件（把一端的条件用到另一端）？
- [ ] 是否把局部结论说成全局结论？
- [ ] 是否忽视奇点（奇点处的行为是否单独分析）？
- [ ] 是否物理上不可接受（能量为负、概率>1、因果违反）？
- [ ] 每个公式是否有文献来源或推导依据？

## 小型评测集（验证流程）

1. **常规二阶ODE**：$y'' + \omega^2 y = 0$，$y(0)=0$，$y(\pi)=0$ → 本征值 $\omega=n$
2. **含奇点方程**：Bessel 方程 $r^2 y'' + r y' + (r^2-n^2)y=0$ → Frobenius 展开
3. **渐近分析**：$\epsilon y'' + y' + y = 0$，$\epsilon\to 0$ → matched asymptotics
4. **本征值问题**：Teukolsky 角向方程 $s=-2$，$a\omega=0.3$ → 球形谐函数退化验证
5. **物理解读**：Kerr $s=-2$ 散射，$k_+<0$ 时 $\mathcal{R}>1$ → 超辐射物理图像
